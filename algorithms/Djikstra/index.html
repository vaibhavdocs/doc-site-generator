    Djikstra

    Djikstra algorithms provides the shortest path between two points and shortest path refers to the cost to reach the endpoint from the source

    There are 2 algorithms that provide the shortest path between vertices i) Breadth first search - Gives the shortest path betweeen verrtices based on no of vertices in betweeen them ii) Djikstra - Shortest path based on the minimum cost to reach that the destination
